# Game Mechanics Iteration 1 (DnD 5.5e Base, Core Systems from Synthesis/Tournaments)

## Core Mechanics Overview
**Philosophy**: Standard 5.5e + strategic layers: Mana Uncertainty (risk/reward), Sunlight EP (positioning/resource), Flux Events (environmental strategy). Magic uncertain vs science (mana flux unpredictable).

## 1. Mana Uncertainty Principle (All Magic/Magitech)
**Resolution**: Before casting spell/device: d20 + caster mod (Int/Wis/Cha) vs DC = 10 + spell level*2 (cantrip 10, lvl9 28).
| d20 Result | Effect |
|------------|--------|
| Nat 20 | Empower: +1d6 dmg/lvl, +50% duration/heal |
| >= DC | Normal |
| < DC | Weak: Half effect, no upcast |
| Nat 1 or <DC-5 | Wild Surge: Roll d100 table (tease below) |

**Wild Surge d100 Tease** (Sample 10): 01-10 Vine Explosion (3d6 pier 10ft), 11-20 Sun Heal ally (2d8), 91-100 Mega Empower (+100%).

**Mitigation**: Flux Stabilizer (rare item, ignore 1/day), Observation (bonus action Arcana DC15 reduce DC-2).

## 2. Sunlight Exposure Points (EP) System
**Tracking**: Personal pool max 20. Gain: 1/turn direct sun (races x2). Night/indoors: 0.
**Spend Table**:
| EP Cost | Effect |
|---------|--------|
| 5 | Advantage next attack/save/skill |
| 10 | Bonus action cast cantrip/spell lvl1 |
| 15 | Reaction Shield (reduce 20 dmg) |
| 20 | Overcharge: Next spell crit auto |

**Strategic**: Day battles favor open fields, night indoors penalize sun-races.

## 3. Flux Meter & Calendar (Environmental)
**Global Flux Meter** (GM tracks 0-20): Starts 10. Weaves/events +/-1-5. >15: Storms (double uncertainty DC), <5: Stable (+prof magic).
**Daily Flux Roll** (d20 city-wide): 1-5 Minor wild, 6-15 Normal, 16-20 Boon (EP+5).

# Aetherial Concord - World History Iteration 1

## Cosmology (Improved)
- **Aetherial Veil**: Mana flows from Solar Heart through planar layers. Uncertainty from Veil fluctuations causes 'Flux Events'.
- **Planes** (detailed):
  | Plane | Description | Magic Influence |
  |-------|-------------|-----------------|
  | Material | Core world, solarpunk cities | Stable mana with Flux risks |
  | Verdant Nexus | Endless forests, time-warped groves | Growth, illusion |
  | Elemental Flux | Crystal seas, lava vines | Elemental crafting |
  | Shadow Veil | Corrupted wastes | Necrotic, fear magics |
  | Celestial Spires | Floating isles of light | Divination, healing |

## Eras (Expanded Timeline)
1. **Dawnweaving (-5000 AR)**: Gods weave reality. Races: Humans, Sylvari (plant elves), Forgeborn (magitech dwarves), Aarakocra, etc.
2. **Arcane Zenith (-1000 to 0 AR)**: Megastructures like Starweave Towers channel mana.
3. **Mana Cataclysm (0 AR)**: Solar Flare rends Veil, empires fall.
4. **Reclamation (0-500 AR)**: Nomad enclaves, first Mana Scavengers.
5. **Magitech Dawn (500-1000 AR)**: Sun Crystals (mana batteries), Vineforged alloys.
6. **Solarpunk Revolution (1000-1250 AR)**: Living cities, bio-arcane harmony. Current threats: Flux Beasts, Ruin Lords awakening.

## Key Historical NPCs/Events
- **Elara Sunweaver (512 AR)**: Discovered first crystal, founded Verdantspire.
- **Vine Accord (800 AR)**: Multi-race treaty for shared magitech.
